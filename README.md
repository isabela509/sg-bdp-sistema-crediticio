# Sistema de Gestión Crediticia SG-BDP

Proyecto de Ingeniería de Software para el Banco de Desarrollo Productivo (BDP).

## Descripción

Sistema diseñado para gestionar el proceso de créditos PYME, incluyendo evaluación financiera, scoring crediticio y aprobación.

## Contenido

- Documentación técnica (docs)
- Diagramas UML (diagrams)
- Prototipo (prototype)

## Tecnologías

- Arquitectura de microservicios
- API Gateway
- Motor de scoring
- Seguridad Zero Trust
